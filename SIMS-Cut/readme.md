# 环境安装

在当前目录运行下属命令， 安装 python 相关环境。
```sh
pip install -r requirements.txt
```

# 使用说明

请编辑 config.py 中相应的项目，之后直接在命令行中运行下述代码即可：

```sh
python SIMSCutpipe.py
```

> 注意，运行该命令之后，会在 process_data_path 目录下生成 process 目录。请确保运行该代码之前 process 目录不存在，不然上述代码会报错。
